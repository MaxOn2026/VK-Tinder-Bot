# VK-Tinder-Bot

**VK-Tinder-Bot** — это программный бот для социальной сети ВКонтакте, который помогает пользователям находить потенциальных партнеров через автоматизированный поиск по заданным критериям. Бот анализирует профили пользователей ВКонтакте, находит подходящие варианты на основе возраста, пола и города, а также предоставляет возможность взаимодействия: добавление в избранные, просмотр фото, чат с мэтчами.

Проект реализован на Python с использованием SQLAlchemy ORM для работы с базой данных PostgreSQL и API ВКонтакте для получения информации о пользователях.

## Обзор

**VK-Tinder-Bot** — это программный бот для социальной сети ВКонтакте, который помогает пользователям находить потенциальных партнеров через автоматизированный поиск по заданным критериям. Бот анализирует профили пользователей ВКонтакте, находит подходящие варианты на основе возраста, пола и города, а также предоставляет возможность взаимодействия: добавление в избранные, просмотр фото, чат с мэтчами.

Проект реализован на Python с использованием SQLAlchemy ORM для работы с базой данных PostgreSQL и API ВКонтакте для получения информации о пользователях.

### Ключевые технологии

- **Python 3.8+** — язык программирования
- **SQLAlchemy 2.0** — ORM для PostgreSQL
- **vk_api** — клиент API ВКонтакте
- **PostgreSQL 12+** — база данных
- **Bot LongPoll API** — реальное время обработки сообщений

## Возможности

- **Автоматизированный поиск партнеров** — поиск потенциальных партнеров по настраиваемым критериям (возраст, пол, город)
- **Галерея фотографий** — просмотр фотографий профилей с сортировкой по популярности (по лайкам)
- **Интерактивный чат** — интерфейс чата в стиле ВКонтакте
- **Управление избранным** — сохранение понравившихся профилей в список избранных
- **Черный список** — блокировка нежелательных профилей
- **Система взаимных мэтчей** — распознавание взаимных симпатий и создание мэтчей
- **Чат между мэтчами** — обмен сообщениями между пользователями, у которых есть взаимный лайк
- **Управление состоянием пользователей** — отслеживание навигации и предпочтений пользователей

## Установка

### Требования

- Python 3.8+
- PostgreSQL 12+
- Токен API ВКонтакте (для бота и пользователя)
- Группа ВКонтакте (для работы бота)

### Шаг 1: Клонирование репозитория

```bash
git clone https://github.com/ваш-аккаунт/VK-Tinder-Bot.git
cd VK-Tinder-Bot
```

### Шаг 2: Создание виртуального окружения

```bash
python -m venv .venv
source .venv/bin/activate  # или .venv\Scripts\activate на Windows
```

### Шаг 3: Установка зависимостей

```bash
pip install -r requirements.txt
```

### Шаг 4: Настройка переменных окружения

Создайте файл `.env` в корне проекта:

```env
# Настройки PostgreSQL
HOST=localhost
PORT=5432
DATABASE=vk_tinder
USER=postgres
PASSWORD=ваш_пароль

# Настройки ВКонтакте
VK_GROUP_TOKEN=ваш_токен_группы
VK_USER_TOKEN=ваш_токен_пользователя
VK_API_VERSION=5.199

# Дополнительные настройки (опционально)
DB_ECHO=false
DB_ECHO_POOL=false
```

#### Получение токенов ВКонтакте

1. **Токен группы**: Создайте группу ВКонтакте и сгенерируйте токен с правами на messages, users и photos

2. **Токен пользователя**: Используйте скрипт `get_token.py` для получения токена пользователя через авторизацию в браузере:

```bash
python get_token.py
```

Это откроет окно браузера для авторизации. Скопируйте токен из терминала и добавьте его в файл `.env`.

### Шаг 5: Создание базы данных

База данных будет создана автоматически при первом запуске бота, или вы можете создать её вручную:

```bash
psql -U postgres -c "CREATE DATABASE vk_tinder;"
```

## API документация

### Модуль базы данных (`database/`)

#### DatabaseManager

Управляет подключениями к PostgreSQL с использованием SQLAlchemy и пула соединений.

```python
from database.db_manager import db_manager

# Инициализация подключения к БД
db_manager.initialize()  # Использует настройки из config.py
# или
db_manager.initialize("postgresql://user:pass@host:5432/dbname")

# Создание таблиц
db_manager.create_tables()

# Использование сессии с контекстным менеджером
with db_manager.get_session() as session:
    users = session.query(BotUser).all()

# Выполнение сырого SQL
results = db_manager.execute_raw("SELECT * FROM users WHERE city = %(city)s", {"city": "Москва"})

# Закрытие подключений
db_manager.close()
```

**Методы:**

- `initialize(db_url: Optional[str] = None) -> None` — Инициализирует подключение к базе данных
- `create_tables(drop_first: bool = False) -> None` — Создает все таблицы базы данных
- `get_session() -> Generator[Session, None, None]` — Получает сессию базы данных
- `execute_raw(query: str, params: dict | None = None) -> Any` — Выполняет сырой SQL
- `close() -> None` — Закрывает все подключения к базе данных

- `age_max`: int — Максимальный возраст для мэтчей
- `max_distance`: int — Максимальное расстояние (км)

- `last_active`: date — Дата последней активности

- `last_name`: str — Фамилия
- `birth_year`: int — Год рождения

- `photo_urls`: list[str] — Массив URL фотографий
- `relation`: int — Семейное положение
- `education`: str — Образование

**UserInteraction** — Записывает действия пользователей над профилями.

- `user_id`: int — ID пользователя, выполнившего действие

- `action`: str — Тип действия (view, like, dislike, block)

**Match** — Представляет мэтч между двумя пользователями.

- `user1_id`: int — ID первого пользователя

**Message** — Представляет сообщение чата.

- `match_id`: int — ID мэтча
